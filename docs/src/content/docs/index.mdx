## Features

<CardGrid>
  <Card title="Text to images">
    Render musical scores via [Markdown](/guides/markdown/) or with an Astro [component](/guides/component/).
  </Card>
  <Card title="Easy setup">
    Works with `remark`, `rehype` and `satteri` Markdown processors out-of-the-box.
  </Card>
  <Card title="Zero client-side JavaScript">
    Built for static sites. Images are compiled at build time, keeping your site fast.
  </Card>
  <Card title="Accessible">
    Automatic `alt` text from the score's title and composer. (Or supply your own.)
  </Card>
</CardGrid>

You can notate lots of different music with LilyPond! Check out the [examples](/examples/).
