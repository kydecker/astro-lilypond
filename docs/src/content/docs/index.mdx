`astro-lilypond` is an [Astro](https://astro.build) integration that lets you write [LilyPond](https://lilypond.org) inside Markdown...

````md
```lilypond
  \relative c' {
    c4 d e f |
    g a b c  \bar "|."
  }
```
````

...and render it as music notation. ♫꒰･‿･๑꒱

```lilypond
\relative c' {
  c4 d e f |
  g a b c  \bar "|."
}
```

## Features

<CardGrid stagger>
  <Card title="Text to SVG or PNG">
    Render LilyPond via [Markdown](/usage/#markdown) or with a `<LilyPond>` [component](/usage/#component).
  </Card>
  <Card title="Easy setup">
    Works with both `unified()` and `satteri()` Markdown processors out-of-the-box.
  </Card>
  <Card title="Zero JS required">
    Images are compiled at build time, keeping your site fast.
  </Card>
  <Card title="Full LilyPond support">
    Works with all LilyPond musical and layout syntax.
  </Card>
</CardGrid>
