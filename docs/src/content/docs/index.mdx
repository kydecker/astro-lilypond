`astro-lilypond` is an [Astro](https://astro.build) integration that lets you write [LilyPond](https://lilypond.org) markup in your Markdown files...

````md
```lilypond
\relative c' {
  \clef treble
  c4 d e f |
  g a b c  |
  c b a g  |
  f e d c \bar "|."
}
```
````

...and render it as music notation. ♫꒰･‿･๑꒱

```lilypond
\relative c' {
  \clef treble
  c4 d e f |
  g a b c  |
  c b a g  |
  f e d c \bar "|."
}
```

Music can be rendered as an `.svg` or `.png`.

In addition to rendering in Markdown, the [`<LilyPond>` component](/usage/#component) lets you import `.ly` files..

...and render them anywhere you use Astro. ♪~
