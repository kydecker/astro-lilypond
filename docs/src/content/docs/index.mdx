`astro-lilypond` is an [Astro](https://astro.build) integration that lets you write [LilyPond](https://lilypond.org) inside Markdown...

````md
```lilypond
  \relative c' {
    c4 d e f |
    g a b c  \bar "|."
  }
```
````

...and render it as music notation. ♫꒰･‿･๑꒱

```lilypond
\relative c' {
  c4 d e f |
  g a b c  \bar "|."
}
```

Music can be rendered as an `.svg` or `.png`.

<LinkCard href="/getting-started" icon="rocket" title="Get Started" description="Install astro-lilypond and render music notation on your site." />

In addition to rendering in Markdown, the [`<LilyPond>` component](/usage/#component) lets you import `.ly` files and render them anywhere you use Astro.
