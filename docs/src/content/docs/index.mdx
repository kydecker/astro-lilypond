`astro-lilypond` is an [Astro](https://astro.build) integration for displaying musical notation on the web.

<HowToUse />

## Features

<CardGrid stagger>
  <Card title="Text to SVG or PNG">
    Render musical scores via [Markdown](/usage/#markdown) or with a `<LilyPond>` [component](/usage/#component).
  </Card>
  <Card title="Easy setup">
    Works with both `unified()` and `satteri()` Markdown processors out-of-the-box.
  </Card>
  <Card title="Ships zero JS">
    Images are compiled at build time, keeping your site fast.
  </Card>
  <Card title="Full LilyPond support">
    Works with all LilyPond display syntax.
  </Card>
</CardGrid>
