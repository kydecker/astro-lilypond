To display music on your site, you will need to install Astro, LilyPond, and this integration.

## Installation

<Steps>

1. [Create an Astro project](https://docs.astro.build/en/install-and-setup/) running `v7.0` or later.

   <InstallAstro />

2. [Install LilyPond](https://lilypond.org/download.html).

   The LilyPond binary is required in order to compile music notation. Make sure it is installed on the computer or container used to `build` your website (locally, in GitHub workflows, etc).

   <InstallLilyPond />

   Rendered scores are written to `public/_lilypond/` (see [`outputDir`](/configuration/#outputdir)). Add this directory to your `.gitignore` — or commit it if you'd rather your deploy environment not need LilyPond installed for builds where nothing changed.

3. Install `astro-lilypond`.

   <InstallAstroLilyPond />

4. Add the integration to your Astro config.

   ```diff lang="js"
     // astro.config.mjs
     import { defineConfig } from 'astro/config';
   + import lilypond from 'astro-lilypond';

     export default defineConfig({
   +   integrations: [lilypond()],
     });
   ```

   See [Configuration](/configuration/) for available options.

5. Start writing music! See [Usage](/usage/) for Markdown and component-based approaches.

</Steps>
