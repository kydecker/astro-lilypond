Here's how to get the `astro-lilypond` integration set up so you can begin writing and displaying notation on your site.

## Installation

<Steps>

1. [Create an Astro project](https://docs.astro.build/en/install-and-setup/) running `v7.0` or later.

   <InstallAstro />

2. [Install LilyPond](https://lilypond.org/download.html).

   The LilyPond binary is required in order to compile music notation. Make sure it is installed on the computer or container used to `build` your website (locally, in GitHub workflows, etc).

   <InstallLilyPond />

3. Install `astro-lilypond`.

   <InstallAstroLilyPond />

4. Add the integration to your Astro config.

   ```diff lang="js"
     // astro.config.mjs
     import { defineConfig } from 'astro/config';
   + import lilypond from 'astro-lilypond';

     export default defineConfig({
   +   integrations: [lilypond()],
     });
   ```

   See the [configuration reference](/reference/configuration/) for available options.

   :::caution

   The `astro-lilypond` extension is intended for use on static sites and has not been tested with server-side rendering. Use caution if you have [`output: 'server'`](https://docs.astro.build/en/reference/configuration-reference/#output) in your Astro config or [`prerender = false`](https://docs.astro.build/en/reference/routing-reference/#prerender) in your file.

   :::

5. Start writing music! You can do a lot with LilyPond; view the [examples](/examples/). Write your LilyPond code directly in [Markdown](/guides/markdown/), or render `.ly` files with a [component](/guides/component/).

</Steps>
