## Prerequisites

1. The LilyPond binary must be available on `$PATH`.
   - [Install the LilyPond binary](https://lilypond.org/download.html), or run `brew install lilypond` (Mac) or `apt install lilypond` (Linux).
   - Verify the install with `lilypond --version`.
2. You should have an `astro` project running `v7.0` or newer.
   - [Install Astro](https://docs.astro.build/en/install-and-setup/).

## Installation

<Steps>

1. Install `astro-lilypond`.

   <Tabs>
     <TabItem label="pnpm">
       ```sh
       pnpm add astro-lilypond
       ```
     </TabItem>
     <TabItem label="npm">
       ```sh
       npm install astro-lilypond
       ```
     </TabItem>
     <TabItem label="yarn">
       ```sh
       yarn add astro-lilypond
       ```
     </TabItem>
   </Tabs>

2. Add the integration to your Astro config.

   ```diff lang="js"
     // astro.config.mjs
     import { defineConfig } from 'astro/config';
   + import lilypond from 'astro-lilypond';

     export default defineConfig({
   +   integrations: [lilypond()],
     });
   ```

   See [Configuration](/configuration/) for available options.

3. Start writing LilyPond. See [Usage](/usage/) for Markdown and component-based approaches.

</Steps>
