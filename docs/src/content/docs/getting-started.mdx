## Prerequisites

<Steps>

1. [Install LilyPond](https://lilypond.org/download.html).

   The LilyPond binary is required in order to compile music notation. Make sure it is installed on the computer or container used to `build` your website (locally, in GitHub workflows, etc).

   <InstallLilyPond />

2. [Install Astro](https://docs.astro.build/en/install-and-setup/) running `v7.0` or later.

   <InstallAstro />

</Steps>

## Installation

<Steps>

1. Install `astro-lilypond`.

   <InstallAstroLilyPond />

2. Add the integration to your Astro config.

   ```diff lang="js"
     // astro.config.mjs
     import { defineConfig } from 'astro/config';
   + import lilypond from 'astro-lilypond';

     export default defineConfig({
   +   integrations: [lilypond()],
     });
   ```

   See [Configuration](/configuration/) for available options.

3. Start writing music! See [Usage](/usage/) for Markdown and component-based approaches.

</Steps>
